## Standard Procedures

**Pre-check-in checklist (all villas):**
1. Pool: clean + correct chemical levels
2. Aircon: tested in all rooms
3. Welcome pack: water, fruit, toiletries stocked
4. WiFi password card visible
5. Emergency contact card visible
6. Gate/door access confirmed

**Maintenance escalation path:**
1. Guest reports issue → log in system (urgent if blocking stay)
2. Notify PIC via Maintenance TVM WhatsApp group
3. If no response within 2 hours → call PIC directly
4. If PIC unavailable → escalate to owner Afni

**Check-out inspection:**
1. Inventory check against move-in record
2. Pool condition
3. Damage report if needed
4. Deep clean scheduled same day
